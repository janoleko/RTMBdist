# AD-compatible error function and complementary error function

AD-compatible error function and complementary error function

## Usage

``` r
erf(x)

erfc(x)
```

## Arguments

- x:

  vector of evaluation points

## Value

`erf(x)` returns the error function and `erfc(x)` returns the
complementary error function.

## Examples

``` r
erf(1)
#> [1] 0.8427008
erfc(1)
#> [1] 0.1572992
```
